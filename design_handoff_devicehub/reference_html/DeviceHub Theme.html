Design system

DeviceHub theme

A green skin for shadcn/ui, built from your colorhunt palette. The structure stays stock shadcn — same tokens, radius, components — so anything from the shadcn ecosystem drops in. We only redefined the colors. The four palette swatches anchor the light theme; dark mode is a separately tuned deep-pine set so it never goes neon or low-contrast.

Color rules

How a 4-color palette becomes a full UI theme

Four swatches can't cover every UI state. These rules turn them into a consistent, accessible system.

1

One brand hue, one ramp

The three greens are points on a single 50–950 ramp. Everything brand-colored samples that ramp — no ad-hoc greens invented per screen.

2

Neutrals carry the UI

Backgrounds, cards, borders and muted text are a near-neutral gray with a whisper of the brand hue. Green stays a deliberate accent, not wallpaper.

3

Contrast decides the fill

#2FA084 on white is 3.24:1 (AA-large only). Light primary uses green-600 with white text for brand identity; swap to green-800 (6:1) where strict body-text AA is required.

4

Dark mode flips the brand

Dark uses a lightened mint (#6FCF97) fill with dark pine text (9:1) — never dark fill + white text. Surfaces are deep pine, chroma dialled down.
